Visão Geral

O que é o BotTrading?

Um sistema de trading automatizado para criptomoedas que utiliza inteligência artificial para detectar regimes de mercado e tomar decisões de compra e venda de forma autônoma.

O projeto conta com paper trading para simulação segura, análise pré-trade com IA e um dashboard React para monitoramento em tempo real. Atualmente em fase de desenvolvimento e testes.

Funcionalidades

O que o projeto entrega

🤖

Trading Bot

Bot automatizado em Python que executa operações de compra e venda em exchanges de criptomoedas com estratégia configurável.

📊

Dashboard React

Interface web construída em React para monitoramento de posições, performance e operações em tempo real.

🧠

Detecção de Regime

Algoritmo de IA que identifica regimes de mercado (tendência, lateralização, volatilidade) para ajustar a estratégia automaticamente.

🔍

Análise Pré-Trade

Sistema de análise inteligente que avalia condições de mercado antes de executar cada operação, reduzindo falsos sinais.

📋

Paper Trading

Modo de simulação que permite testar estratégias com dados reais de mercado sem arriscar capital real.

🔗

Integração Kraken

Conectado à API da Kraken para execução de ordens e obtenção de dados de mercado em tempo real.
